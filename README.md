# Store Manager

A **full-stack Store Management application** built with **Next.js**, **Prisma**, and **PostgreSQL**. This app allows users and store owners to manage stores, submit ratings, and view analytics.

---

## Features

- **User Management**: Register, login, and role-based access (Admin, Store Owner, User).  
- **Store Management**: Admins can add stores and view all stores.  
- **Ratings**: Users can submit ratings for stores.  
- **Dashboard**: Admin dashboard with analytics (total users, stores, ratings).  
- **Role-based Access**: JWT-based authentication and middleware for protected routes.

- **Authentication**: JWT  
- **State Management**: React useState / useContext
